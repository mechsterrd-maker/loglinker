## What's built

### Tabs (visible)
- **Home** — Plant Pulse, alerts, quick stats, plant info
- **Production** — Machines, dies, shots (with cascade engine)
- **Quality** — NCRs (auto-raised when reject % > 3%)
- **Chat** — Real-time messaging
- **More** — Navigator to all other modules

### Tabs (under More)
- **People** — Units, team, shifts, invite codes
- **Maintenance** — Breakdowns, machine status
- **Stocks** — Items, GRN, issues, transactions
- **Action Hub** — All open commitments

### Cascade triggers (database-level)
- Shot logged → die stroke count auto-increments
- Reject % > 3% → NCR auto-raised with severity
- Breakdown reported → machine status → 'breakdown'
- Breakdown resolved → machine → 'running' + downtime calculated
- Stock txn → current_qty auto-updated

### RLS (re-enabled, properly)
- `my_plant_id()` SECURITY DEFINER function breaks recursion
- Every table has `*_all` policy scoped to current user's plant
- Special policies for users (self + plant), plants (read own + insert any for signup)

### Audit log
- All key tables (shots, NCRs, breakdowns, stock txns, units, users, shifts) have audit triggers
- Visible in More → Audit Log (plant_head and admin only)

### Invite redemption
- `redeem_invite_code(code, name, phone)` RPC
- Wizard now has 3 paths: Choose → New Plant or Join with Code
- Code is auto-generated by Plant Head in People → Team → Invite
