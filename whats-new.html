Layout strategy
Waste-first packing
Before opening a new sheet the planner first scans every ring hole on every already-placed ring. Only when nothing fits in the existing waste does it extend.
Recursion
Rings inside rings
Small ring nests in big ring's hole, small disc nests in the small ring's hole. Each level checks dimension fit + 80% area budget.

5 Chat — ultra-fast

Two real bottlenecks killed.

Sidebar
Instant in-place update
New message arrives → unread badge + last-message preview update within one frame. The old 1.5s debounced full reload is gone.
Startup
Lazy-mount tabs
Every module used to mount on app start — ~80 requests in one burst. Now only the active tab mounts. Click-triggered requests get a free connection.
Voice notes on Mac
Honest playback
Android Chrome's opus-in-mp4 can't decode on Safari. The audio element now flips to a clean "🎤 tap to download" chip instead of a stuck 00:00 player.
Send button
Always visible
→ button never disappears. Mic gets its own slot beside it. No more "where did send go?"
