5 Voice notes

10× cheaper transcription, browser-honest recording, working Mac playback.

Transcription #41
Groq Whisper + silence trimming
Replaces the old OpenAI Whisper path. Silence at start/end is auto-trimmed before upload. ~10× cheaper per minute.
Recorder MIME #93
Requires explicit AAC for mp4
Android Chrome answers true to audio/mp4 but actually emits opus-in-mp4 — Safari then silently rejects. Now we require codecs=mp4a.40.2 before picking mp4; otherwise honest audio/webm.
Mac playback fallback #58 #93
Download chip on undecodable codecs
Audio element flips to "🎤 Voice note — this browser can't play this codec — tap to download" when metadata reports 0 duration or onError fires.
Recording UI #75
Stop button can't be hidden
During recording, the toolbar buttons hide so ⏹ has guaranteed room on narrow mobile.
