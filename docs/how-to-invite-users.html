Adding a User & Joining with an Invite Code

A short guide for plant admins and the team members they invite
Loglinkr · v1
loglinkr.com
Loglinkr uses an invite-code system to add team members safely. The admin creates an invite from the People tab; the new user signs up using the code (or a single-tap WhatsApp link) and is automatically attached to your plant with the right role and unit.

What's inside

  1. Part 1 — Admin: creating an invite (4 steps)
  2. Part 2 — User: joining the plant (3 steps)
  3. Roles & what each one can do
  4. Troubleshooting & common questions

1Admin · Creating an Invite

You'll need the new person's name and mobile number. Email is optional but recommended. Takes about 30 seconds.

Step 1 · Open Loglinkr and go to the People tab

Sign in to loglinkr.com/app on your phone or computer. From the bottom (or top) menu, tap People.

📱
Screenshot 1 — People tab in the bottom menu
Open Loglinkr → tap the People icon in the navigation bar. The People page lists the current team and pending invites.

Step 2 · Tap "+ Invite" in the Team section

Under the Team section header you'll see the count of current members and a + Invite button on the right. Tap it — the invite form opens.

➕
Screenshot 2 — Team section with the "+ Invite" button
"Team · 5" (or whatever your count is) with a small + Invite button on the right. Tap to open the invite form.

Step 3 · Fill in the details

You'll see this form:

Full Name *
The person's full name as you want it shown in the app — e.g. Selvam K.
Phone *
10-digit Indian mobile, no country code (the app adds +91 automatically). This is how they'll be reached on WhatsApp.
Email (optional but recommended)
If provided, the user will sign up with email + password. If skipped, they can sign up with their phone.
Role
Determines what they can see and do. See the roles section below.
Primary Unit
Which plant unit this user is mostly based at (only relevant for multi-unit plants).
📝
Screenshot 3 — Invite form filled in
Example: Selvam K · 9876543210 · selvam@example.com · Supervisor · Unit 1. Tap Send Invite at the bottom to create the code.

Step 4 · Share the invite

Loglinkr generates a 6-character invite code instantly and shows it like this:

Invite code
A7K3B9

You have three ways to deliver it:

  • Send via WhatsApp (green button) — opens WhatsApp with a pre-filled message and the invite link. Recommended.
  • Copy link — copies a sign-up URL like loglinkr.com/app?code=A7K3B9. Paste it anywhere.
  • Read out the code — they can type it manually during signup (last resort).
💬
Screenshot 4 — "Invite created" panel with WhatsApp / Copy link buttons
Big invite code on top, copy-link box, and a green Send via WhatsApp button. The link goes to the same place; the code is just a fallback.
Tip: The link is the easiest way. The user taps it on their phone, fills email + password, and they're in — no code to remember.

2Roles · What Each Can Do

Choose the lowest role that fits — you can always promote them later. Role gates what they see; it does not block them from being on the team.

Operator
Floor staff. Logs production, posts in chat, marks tasks done. No admin access.
Supervisor
Shift lead. Can verify entries, raise NCRs, manage their crew. Read-only on commercial data.
Manager
Unit-level admin. Stocks, schedules, billing, vendors, projects. Cannot edit plant settings.
Plant Head
Full access. Plant settings, masters, IATF/audit, billing, all units. Can invite & remove users.

3User · Joining the Plant

This is the section to forward to the person you invited. Three steps, about a minute.

Easy way · Tap the invite link

  1. Tap the link your admin sent on WhatsApp (looks like loglinkr.com/app?code=A7K3B9).
  2. Loglinkr opens and shows a sign-up screen with the plant name already detected. Enter your email and a password (or your phone number).
  3. Tap Sign up — you're automatically added to the plant. The app opens on Home.
🔗
Screenshot 5 — Invite landing screen
Welcome panel: "You're being added to [Plant name] as [role]." with the email + password fields underneath and a green Sign up button.

Manual way · Type the code

If you only have the 6-character code, no link:

  1. Open loglinkr.com/app in any browser.
  2. Tap Sign up (or I have an invite code).
  3. Type your name, email, password — and paste the invite code in the field provided.
  4. Tap Sign up. Loglinkr verifies the code and attaches you to the plant.
⌨️
Screenshot 6 — Signup form with invite-code field
Standard sign-up form with an extra Invite code field at the bottom. Enter the 6-character code your admin gave you.

Install the app (recommended)

Once you're signed in, install Loglinkr on your phone so it opens like any normal app — no browser bar, push notifications work.

📱 Android (Chrome)

Three-dot menu → Install app → confirm. Loglinkr icon appears on your home screen.

🍎 iPhone (Safari)

Share button → Add to Home Screen → Add. Loglinkr icon appears on your home screen.

4Troubleshooting & Questions

"Invite code is invalid or already used."
Each code works only once. If the user already signed up, they should just sign in with their email instead. If the code was never used, ask the admin to create a fresh invite — codes can be revoked or expire.
The user didn't get the WhatsApp message
Open the Pending Invites section on the People tab → tap the user's row → re-share. The same link/code is still valid.
"User limit reached" when creating an invite
Your plan has a user cap (Pro = 20 users, including pending invites). Either remove an unused invite, deactivate a user, or upgrade the plan from More → Subscription.
User signed up but doesn't see the plant
They probably signed up without using the invite link/code. Ask them to sign out, then use the invite link (or paste the code on signup). The link auto-attaches them; bare signup creates an empty new account.
Can I change someone's role later?
Yes. People tab → tap the user's row → change role → Save. Plant Head and Manager roles can do this; Operators and Supervisors cannot.
Can I remove a user?
People tab → tap the user → Deactivate. Their data (entries, tasks, etc.) stays for audit; only their login is disabled. You can reactivate later.
Security note: Never share an invite code in a public group. The first person to use it gets attached to your plant — treat it like a one-time password.
